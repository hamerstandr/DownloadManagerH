# 🧩 افزونه مرورگر Chrome برای دانلود منجر حامد

این افزونه به شما اجازه می‌دهد با راست‌کلیک روی هر لینک، آن را مستقیماً به برنامه دانلود منجر حامد ارسال کنید 🚀

## راهنمای نصب افزونه Chrome

1. ابتدا فایل‌های افزونه را در پوشه `ChromeExtension` قرار دهید.
2. مرورگر Chrome را باز کنید و به آدرس زیر بروید:
   - `chrome://extensions`
3. گزینه **Developer mode** (حالت توسعه‌دهنده) را در بالای صفحه فعال کنید.
4. روی دکمه **Load unpacked** (بارگذاری افزونه بدون بسته‌بندی) کلیک کنید.
5. پوشه `ChromeExtension` را انتخاب کنید.
6. افزونه با آیکون برنامه فعال می‌شود و آماده استفاده است.

### استفاده از افزونه
- روی هر لینک راست‌کلیک کنید و گزینه **ارسال به دانلود منجر حامد 🚀** را انتخاب کنید.
- لینک به برنامه ارسال می‌شود و دانلود آغاز خواهد شد.

### نکته
- برای ارتباط افزونه با برنامه، باید Native Messaging یا پروتکل سفارشی (مانند `downloadmenger://`) در برنامه فعال باشد.
