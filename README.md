# دانلود منجر حامد (DownloadManagerH)

<div dir="rtl" align="right">

## 📖 درباره پروژه
**دانلود منجر حامد** یک نرم‌افزار مدیریت دانلود پیشرفته، سبک و متن‌باز است که با تکنولوژی .NET 9 و WPF توسعه یافته است. این برنامه با تمرکز بر سرعت، امنیت و تجربه کاربری طراحی شده و قابلیت یکپارچه‌سازی کامل با داشبورد مانیتورینگ **TrafficWatch** را دارد.

### ✨ ویژگی‌های کلیدی
- **🌐 چندزبانه**: پشتیبانی همزمان از زبان‌های **فارسی**، **عربی** و **انگلیسی** با تغییر جهت خودکار رابط کاربری (RTL/LTR).
- **🚀 موتور دانلود قدرتمند**: پشتیبانی از دانلود چندبخشی (Multi-threading) برای حداکثر سرعت.
- **🔗 یکپارچگی با مرورگر**: افزونه‌های اختصاصی برای Chrome، Edge و Firefox.
- **📊 مانیتورینگ زنده**: اتصال به TrafficWatch برای نمایش آمار لحظه‌ای در داشبورد وب.
- **🎨 رابط کاربری مدرن**: طراحی ریسپانسیو، تم تاریک/روشن و تجربه کاربری روان.
- **⚙️ تنظیمات پیشرفته**: جستجو در تنظیمات، محدودیت پهنای باند، زمان‌بندی دانلود.

## 📥 نصب و راه‌اندازی

### پیش‌نیازها
- **.NET Desktop Runtime 9.0** یا بالاتر.
- سیستم عامل: ویندوز ۱۰/۱۱.

### روش کامپایل از سورس
1. مخزن را کلون کنید:
   ```bash
   git clone https://github.com/hamerstandr/DownloadManagerH.git
   cd DownloadManagerH
   ```
2. پروژه را بیلد کنید:
   ```bash
   dotnet build -c Release
   ```
3. فایل اجرایی در مسیر `bin/Release/net9.0-windows/` قرار دارد.

## 🌍 تغییر زبان برنامه
برای تغییر زبان برنامه:
1. به بخش **تنظیمات (Settings)** بروید.
2. در قسمت **General**، گزینه **Language** را پیدا کنید.
3. یکی از زبان‌های **Persian (فارسی)**، **Arabic (العربية)** یا **English** را انتخاب کنید.
4. برنامه به‌صورت خودکار جهت چیدمان (راست‌چین/چپ‌چین) را تنظیم می‌کند.

---

## 🔌 افزونه‌های مرورگر
این برنامه دارای افزونه‌های اختصاصی برای مدیریت دانلود از طریق مرورگر است.

| مرورگر | وضعیت | مسیر فایل‌ها |
|--------|-------|-------------|
| **Google Chrome** | ✅ آماده | `DownloadManager/Extensions/Chrome/` |
| **Microsoft Edge** | ✅ آماده | `DownloadManager/Extensions/Edge/` |
| **Mozilla Firefox** | ✅ آماده | `DownloadManager/Extensions/Firefox/` |

### نحوه نصب افزونه (بدون نیاز به بسته‌بندی)
نیازی به زیپ کردن فایل‌ها نیست. مراحل زیر را طی کنید:

#### برای کروم و اج:
1. مرورگر را باز کنید و به آدرس `chrome://extensions` (یا `edge://extensions`) بروید.
2. گزینه **Developer mode** را در گوشه بالا فعال کنید.
3. روی دکمه **Load unpacked** کلیک کنید.
4. پوشه مربوطه (مثلاً `DownloadManager/Extensions/Chrome`) را انتخاب کنید.

#### برای فایرفاکس:
1. به آدرس `about:debugging#/runtime/this-firefox` بروید.
2. روی **Load Temporary Add-on** کلیک کنید.
3. فایل `manifest.json` داخل پوشه `DownloadManager/Extensions/Firefox` را انتخاب کنید.

> **نکته:** افزونه‌ها به‌صورت خودکار با برنامه اصلی روی پورت `9090` ارتباط برقرار می‌کنند.

## 📡 یکپارچه‌سازی با TrafficWatch
این برنامه می‌تواند به عنوان یک گره (Node) به داشبورد **TrafficWatch** متصل شود و آمار دانلودها را گزارش دهد.

### فعال‌سازی
1. در برنامه دانلود منجر، به **تنظیمات > یکپارچگی شبکه** بروید.
2. تیک **فعال‌سازی سرویس TrafficWatch** را بزنید.
3. پورت پیش‌فرض `9090` است (در صورت تداخل تغییر دهید).

### اندپوینت‌های API
وقتی سرویس فعال است، اطلاعات زیر در دسترس قرار می‌گیرند:

- `GET http://127.0.0.1:9090/api/status`: وضعیت کلی برنامه.
- `GET http://127.0.0.1:9090/api/downloads/active`: لیست دانلودهای فعال.
- `GET http://127.0.0.1:9090/api/stats/history`: آمار تاریخی مصرف.

برای جزئیات فنی بیشتر، فایل [`docs/TRAFFIC_WATCH_INTEGRATION.md`](docs/TRAFFIC_WATCH_INTEGRATION.md) را مطالعه کنید.
